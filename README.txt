
#### Data Mining the Diabetes Mellitus disease

A project utilising Data Mining (KDD) and Machine Learning to classify positive or negative diabetes mellitus status in ICU patients using diagnostic data.

Project files include:

* Raw data contained in DiabetesClassificationDataset2022.csv and DataDictionaryCoursework2022.csv. Raw data supplied by UEA CMP School.
* Python script dmproject.py
* requirements.txt
* Project write-up report.pdf

Results can be reproduced by running dmproject.py with the library versions supplied. Downloading raw data not necessary. The script reproduces all visualisations and results tables detailled in the write-up report.
